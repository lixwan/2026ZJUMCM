# 问题2：抽签方案设计与公平性证明

## 一、数据结构与符号约定

### 参赛队伍
- **市级队**：$C_1, C_2, \ldots, C_{11}$（11个副省级/地级市）
- **县级队**：$T_1, T_2, \ldots, T_{53}$（20个县级市 + 33个县）
- **总计**：64支队伍，分入16个小组 $G_1, G_2, \ldots, G_{16}$，每组4队

### 代管关系
记 $S(C_i) = \{T_j \mid T_j \text{ 由 } C_i \text{ 代管}\}$，即市级队 $C_i$ 代管的县级队集合。

| 市级队 | 代管县级队数量 |
|--------|:----------:|
| 杭州 | 3 |
| 宁波 | 4 |
| 温州 | 8 |
| 嘉兴 | 5 |
| 湖州 | 3 |
| 绍兴 | 3 |
| 金华 | 7 |
| 衢州 | 4 |
| 舟山 | 2 |
| 台州 | 6 |
| 丽水 | 8 |

### 约束条件

1. **C1（市级分散）**：$\forall i \neq j,\ G(C_i) \neq G(C_j)$，即任意两个市级队不在同一小组。
2. **C2（代管回避）**：$\forall i, \forall T \in S(C_i),\ G(C_i) \neq G(T)$，即市级队不与代管的县级队同组。
3. **C3（县级分散）**：$\forall i, \forall T_a, T_b \in S(C_i),\ T_a \neq T_b$，尽量使 $G(T_a) \neq G(T_b)$，即同一市代管的县级队尽量分散。

## 二、抽签方案设计

### 2.1 核心思想

采用**分层随机抽样 + 条件拒绝**的方式，确保每一步都在满足约束的条件下均匀随机。

### 2.2 抽签流程

```
第一步：市级队入组
  → 将11个市级队随机分配到11个不同小组（16选11）
  → 剩余5个小组为空（无市级队），记为"自由小组"

第二步：县级队分批入组（按代管市分批处理）
  → 对每个市级队 C_i，将其代管的县级队 S(C_i) 逐一放入签池
  → 每个县级队随机抽取一个"合法小组"（满足约束）
  → "合法小组"定义为：
      - 不与 C_i 同组（C2：代管回避）
      - 不与已入组的同源县级队同组（C3：县级分散，尽量满足）
      - 该组尚未满员（< 4队）
      - 如果无满足C3的合法小组，则放松C3约束

第三步：最终检查
  → 确保所有小组恰好4队
  → 验证所有约束
```

### 2.3 算法的数学形式

设当前已分配队伍的小组容量状态向量为 $\mathbf{g} = (g_1, \ldots, g_{16})$，其中 $g_k \in \{0,1,2,3,4\}$。

对于市级队 $C_i$ 代管的县级队 $T \in S(C_i)$，其可入小组集合为：

$$\mathcal{L}(T) = \{G_k \mid g_k < 4,\ G_k \neq G(C_i)\}$$

若进一步考虑C3约束（县级分散），则：

$$\mathcal{L}^*(T) = \mathcal{L}(T) \setminus \{G_k \mid \exists T' \in S(C_i),\ T' \text{ 已分配},\ G(T') = G_k\}$$

若 $\mathcal{L}^*(T) = \emptyset$，则退回使用 $\mathcal{L}(T)$。

从合法集合中**等概率**随机选取一个小组。

### 2.4 公平性保证

抽签方案的公平性体现在：

1. **对称性**：市级队之间在抽签前完全对称（均为随机分配），县级队在其合法集合内等概率选择。
2. **均匀性**：由于每一步都从合法集合中等概率随机选取，最终每个合法分组方案出现的概率相等（在满足约束的条件下）。
3. **无偏性**：任何一支队伍落入任何一个"合法"小组的概率均等。

## 三、公平性的统计验证

### 3.1 验证指标

对 $N = 10000$ 次模拟抽签：

1. **单队概率分布**：对每支队伍 $X$，统计其落入各组 $G_k$ 的频率 $p_{X,k}$。理想情况下，对于市级队 $C_i$：
   $$p_{C_i,k} = \begin{cases} \frac{1}{16} & \text{若 } G_k \text{ 对 } C_i \text{ 合法} \\ 0 & \text{否则} \end{cases}$$

   对于县级队 $T \in S(C_i)$，在市级队位置确定后，其不被分配到的组仅为 $G(C_i)$（1个组），因此：
   $$p_{T,k} = \frac{1}{15} \quad (\text{对 } G_k \neq G(C_i))$$

2. **卡方 goodness-of-fit 检验**：检验每支队伍在各合法小组间的分布是否均匀。
   $$\chi^2 = \sum_{k} \frac{(O_k - E_k)^2}{E_k} \sim \chi^2(df)$$

3. **分组方案均匀性**：将每次模拟得到的完整分组方案编码，检验所有可能方案是否均匀出现。

### 3.2 假设检验

- $H_0$：队伍落入各合法小组的概率均等（抽签公平）
- $H_1$：概率不均等（存在偏差）
- 显著性水平 $\alpha = 0.05$
